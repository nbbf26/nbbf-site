# NBBF – Fair Play-regler

1. Vis respekt for motspillere, arrangører og dommere.  
2. Ingen trakassering, krenkelser eller dårlig sportsånd.  
3. Habilitet: du deltar ikke i avgjørelser som involverer deg selv.  
4. Konflikter løses rolig og saklig med arrangør.  
5. Ved grove brudd kan spiller bortvises og rapporteres til NBBF.
